* galleries

  * politicians: images range from 532 x 532 to 607 x 607 pixels
  * sixteen_types: all images are 507 x 507 pixels
  * tv_shows: images range from 507 x 507 to 583 x 583 pixels
* random: all images are 627 x 627 pixels
* site: images are scaled down images from the galleries
